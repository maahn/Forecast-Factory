Interactive version of Richardson's forecast factory. Inspired by

Charlton-Perez, A., and H. Dacre, 2011: Lewis fry richardson’s forecast factory – for real. Weather, 66, 52–54, https://doi.org/10/bn9m7k.


Run locally with 
```
git clone https://github.com/maahn/Forecast-Factory
pip install -r requirements.txt
streamlit run forecast_factory.py
```
Run online with
https://maahn-forecast-factory-forecast-factory-ryuyna.streamlit.app/
